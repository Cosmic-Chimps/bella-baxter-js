# Bella Baxter — JavaScript / TypeScript SDK

Monorepo workspace for Bella Baxter JavaScript and TypeScript packages.

## Packages

| Package | Location | Purpose |
|---------|----------|---------|
| `@bella-baxter/client` | `packages/client/` | Auto-generated OpenAPI client (Fetch-based) |
| `@bella-baxter/node` | `packages/node/` | Node.js secrets loader — injects secrets into `process.env` |

## Quick start

```bash
pnpm add @bella-baxter/node
```

```typescript
import { BellaClient } from "@bella-baxter/node";

const client = new BellaClient({
  baxterUrl: process.env.BELLA_BAXTER_URL!,
  apiKey: process.env.BELLA_API_KEY!,
  environmentSlug: process.env.BELLA_ENV ?? "production",
});

// Load all secrets into process.env before starting the server
await client.loadIntoEnv();
```

## Typed Secret Code Generation

`bella secrets generate typescript` fetches the **secrets manifest** (key names + type hints, no values) from the Bella API and generates a typed `AppSecrets` class. Each getter reads from `process.env` at runtime — **no secret values are ever embedded in the generated file**.

```bash
bella secrets generate typescript \
  --project my-app \
  --environment production \
  --output AppSecrets.ts
```

**Generated `AppSecrets.ts`:**

```typescript
// Auto-generated by bella secrets generate typescript — do not edit manually.

export class AppSecrets {
  get databaseUrl(): string {
    const v = process.env["DATABASE_URL"];
    if (!v) throw new Error("Secret 'DATABASE_URL' is not set.");
    return v;
  }

  get port(): number {
    const v = process.env["PORT"];
    if (!v) throw new Error("Secret 'PORT' is not set.");
    return Number(v);
  }

  get enableFeatureX(): boolean {
    const v = process.env["ENABLE_FEATURE_X"];
    if (!v) throw new Error("Secret 'ENABLE_FEATURE_X' is not set.");
    return v === "1" || v.toLowerCase() === "true";
  }
}
```

### Usage

```typescript
// Load secrets into process.env first (via bella run or the Node SDK)
import { AppSecrets } from "./AppSecrets";

const secrets = new AppSecrets();
const dbUrl = secrets.databaseUrl;  // string — throws if missing
const port  = secrets.port;         // number — parsed automatically
```

Because each getter reads `process.env` on every access (not cached), values updated by `bella watch` are reflected immediately in the next access.

### Options

| Option | Default | Description |
|--------|---------|-------------|
| `-p, --project <slug>` | `.bella` context | Project slug |
| `-e, --environment <slug>` | `.bella` context | Environment slug |
| `--provider <slug>` | `default` | Provider slug |
| `-o, --output <path>` | `AppSecrets.ts` | Output file path |
| `--class-name <name>` | `AppSecrets` | Class name |
| `--dry-run` | — | Print to stdout without writing |
